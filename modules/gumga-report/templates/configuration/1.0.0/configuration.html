Configurações adicionais

Atributos

Propriedade Tipo Exemplo Descrição
stimulsoft.datasource.limitPresentationTables STRING|INT 
stimulsoft.datasource.limitPresentationTables=ALL|0
 Limita a quantidade de tabelas exibidas na criação de data source. Quando utilizado ALL é buscado todas as tabelas do banco ao criar data source.
stimulsoft.search.usesObj BOOLEAN 
stimulsoft.search.usesObj=true
 Obriga a utilização do alias obj nas querys dos data sources.

É possível criar propriedades que serão utilizadas como parâmetros nas querys. Para isto, é preciso que a propriedade tenha por prefixo a palavra 'stimulsoft'. ex: stimulsoft.minhapropriedade.usuario = teste. Na documentação do componente gumga-report-ng contém exemplos de como utilizar os parâmetros personalizados.
