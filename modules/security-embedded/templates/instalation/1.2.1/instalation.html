Segurança Embarcado - Instalação

Segurança embutido para aplicações feitas com framework Gumga.


Pré-requisitos Backend

  • Framework Gumga - ^1.5.0
  • Security Gumga - ^1.2.19

Pré-requisitos Frontend

  • angular-sanitize - ^1.5.8
  • angular-ui-bootstrap - ^2.5.0

Instalação em 3 passos

1 - Baixe o gumga-security-embedded via bower 

bower install gumga-security-embedded --save

2 - Após a baixar adicione o arquivo de distribuição. 

<script type="text/javascript" src="bower_components/gumga-security-embedded/dist/security-embedded.min.js"></script>

3 - Adicione o modulo gumga.securityEmbedded na sua aplicação. 

angular.module('yourApp', ['gumga.securityEmbedded'])

Tudo certo, agora você já pode começar a usar o segurança embutido.
