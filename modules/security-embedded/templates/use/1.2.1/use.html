Exemplo de uso

Para começar a usar o segurança embutido basta adicionar o código abaixo em seu html. 

<security-embedded configuration="config"></security-embedded>

É necessário que você informe uma configuração para o security-embedded. 

$scope.config = {
    appURL : 'http://SEU_SITE.EXEMPLO/SUA_API',
    eternalToken: '', // Caso esse atributo não for informado, pegaremos o token da sessionStorage.
    user: true, // Caso não for informado o valor padrão é TRUE.
    perfil: true // Caso não for informado o valor padrão é TRUE.
};
