Gumga Custom Fields


O GumgaCustomFields é um recurso presente no framework que possibilita adicionar campos a uma entidade sem alterar a estruta na tabela do banco de dados. (POR CLIENTE)

Utilização

Para utiliza-lo, é necessário fazer uma simples alteração na entitade desejada, apenas altere a especialização de GumgaModal para GumgaCustomizableModel.

Se você sua aplicação segue os padrões utilizados pelo Gerador, então provavelmente será necessário a implementação do componente no frontend para este recurso. Clique aqui e veja como implementa-lo.

Se você busca mais informações desta ferramenta, clique aqui e veja o JavaDocs completo, abaixo segue um exemplo de utilização do mesmo:


@Entity
@SequenceGenerator(name = GumgaModel.SEQ_NAME, sequenceName = "SEQ_CAR")
@GumgaMultitenancy
// Alteração
public class Car extends GumgaCustomizableModel {
	
	public Car() {
		
	}
	
	public Car(String color) {
		this.color = color;
	}
	
	private String color;

	public String getColor() {
		return color;
	}

	public void setColor(String color) {
		this.color = color;
	}
	

}
