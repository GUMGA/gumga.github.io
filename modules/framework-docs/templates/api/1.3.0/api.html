Api


Nesta camada estão todos os métodos que serão invocados pelo front end via HTTP requests. Estes métodos são mapeados como serviços REST (RESTful Web Services). Portanto, dada a classe de API abaixo:


    @RestController
    @RequestMapping("/api/pessoa")
    public class PessoaAPI extends GumgaAPI<Pessoa, Long> {


        @Autowired
        public PessoaAPI(GumgaService<Pessoa, Long> service) {
            super(service);
        }
    }

Gerando a camada de API

Entre no modulo api e execute a seguinte linha de comando no Terminal do Linux:


	mvn io.gumga:gumgag:api
        -Dentidade=br.com.exemplo.domain.model.Pessoa
