Estrutura do projeto


O Gumga Framework é composto por várias camadas, cada uma com uma responsabilidade dentro do sistema. São elas:

  • Domain
  • Infrastructure
  • Application
  • API
  • Gateway
  • Presentation

Gerando a estrutura do projeto

Execute a seguinte linha de comando no Terminal do Linux:


    mvn archetype:generate
        -DinteractiveMode=false
        -DarchetypeGroupId=io.gumga
        -DarchetypeArtifactId=gumga-archetype
        -DarchetypeVersion=LATEST
        -DgroupId=br.com
        -DartifactId=exemplo
        -Dversion=1.0.0

Você pode adicionar o archetype da Gumga na IDE de sua preferência.

  • Intellij IDEA
  • Eclipse
