Entidade


A camada domain compreende as entidades do sistema e é nela que configuramos os mapeamentos JPA. Lembrando que o framework de persistência utilizado é o Hibernate. Porém devemos ressaltar que NÃO é nesta camada que configuramos o persistence.xml

    @GumgaMultitenancy
    @SequenceGenerator(name = GumgaModel.SEQ_NAME, sequenceName = "SEQ_Pessoa")
    @Audited
    @Entity
    @Table(name = "Pessoa")
    public class Pessoa extends GumgaModel<Long> {

        @Version
        private Integer version;
        @Column(name = "nome")
        private String nome;

        public Pessoa() {
        }

        public String getNome() {
            return nome;
        }

        public void setNome(String nome) {
            this.nome = nome;
        }
    }

Gerando a classe de entidade

Entre no modulo de domínio e execute a seguinte linha de comando no Terminal do Linux:

            
    mvn io.gumga:gumgag:entidade
        -Dentidade=br.com.exemplo.domain.model.Pessoa
        -Datributos="nome:String"

    mvn io.gumga:gumgag:entidade
        -Dentidade=br.com.exemplo.domain.model.Grupo
        -Datributos="descricao:String,pessoas:List<Pessoa>:@OneToMany"

Parâmetro entidade: Você precisa passar o caminho do pacote que deseja adicionar a entidade.

Parâmetro atributo:Para separar os parâmetros do atributo utilizar dois pontos ( : ) e para separar os atributos utilizar vírgula ( , )

  1. Nome do atributo
  2. Tipo do atributo

  3. (Opcional) Associação se houver

    Associações possiveis:

    • @ManyToOne
    • @ManyToMany
    • @OneToOne
    • @OneToMany
