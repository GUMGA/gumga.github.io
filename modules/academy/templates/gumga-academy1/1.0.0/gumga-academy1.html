G.A. I - Mãos a obra (8 horas)


O que são, para que servem no desenvolvimento GUMGA e como instalar:


  • Java 8, Maven 3
  • NodeJS, Npm, Bower
  • Como utilizar o Archetype GUMGA e criar um projeto
  • Como criar uma entidade, e demais camadas do projeto com o GUMGAG (gerador de código de linha de comando)
  • Utilizando IDEs e ferramentas com o projeto, (Netbeans, http-server, etc…)
  • Configuração de banco de dados
  • Conceitos básicos do segurança, como cadastrar um software e operações, usuário, organização, instância e perfil e utilizá-lo no projeto criado
  • Testando a API através de ferramentas (postman, firefox, etc..)
  • Apresentar conceitos do framework relativos a Multitenancy
  • Oficina prática com entrega de projeto no github
