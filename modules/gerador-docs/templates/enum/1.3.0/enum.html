Gerando enum

Entre no modulo de domínio e execute a seguinte linha de comando no Terminal do Linux:

            
    mvn io.gumga:gumgag:enum
        -Denum=br.com.exemplo.domain.model.TipoDePessoa
        -Dvalores="FISICA,JURIDICA"
