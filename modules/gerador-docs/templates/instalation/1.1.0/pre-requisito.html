Instalação


Pré-requisitos
  • Java 8
  • Maven 3
  • Bower
  • NPM

1
npm install -g gumga-cli @angular/cli

Primeiro, instale o Node.js. Em seguida, instale o gumga cli.
Depois, você está pronto para executar as linhas de comando em seu terminal.

2
gg init myApp

Cria uma nova aplicação usando nossos modelos prontos.
Siga as instruções do gerador, será possível escolher a tecnologia que você deseja usar.

3
cd myApp 
gg build

Faça a compilação da sua aplicação. Nesse momento baixaremos todas as dependência necessárias do seu projeto.
