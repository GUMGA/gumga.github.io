Gumga Components

Os componentes listados aqui, são os mesmos utilizados no Gumga Framework.


Pré-requisitos

  • AngularJS - 1.5.8
  • Angular UI Bootstrap - 2.4.0
  • ngImgCrop (A partir da versão 3.0.0 deve-se adicionar dependências da lib ng-img-crop ao projeto, esta lib é usada no componente GumgaImageUpload, caso esteja usando a versão 2.11.6 e ocorra erro na injeção de dependência, também adicione a lib ng-img-crop ao projeto).
  • JQuery - 2.1.4
  • Bootstrap Notify - 3.1.3

Instalação

Para realizar a instalação execute o comando abaixo na pasta raiz do seu projeto.

bower install gumga-components --save

Adicione o script dos componentes em seu HTML:

<script src="bower_components/ng-img-crop/compile/minified/ng-img-crop.js"></script>
<script src="bower_components/angular-bootstrap/ui-bootstrap-tpls.min.js"></script>
<script src="bower_components/remarkable-bootstrap-notify/dist/bootstrap-notify.min.js"></script>
<script src="bower_components/jquery/dist/jquery.min.js"></script>
<script src="bower_components/gumga-components/dist/gumga.min.js"></script>

Também é necessário adicionar o css em seu HTML:

<link rel="stylesheet" href="bower_components/gumga-components/dist/gumga.min.css"/>

Logo após a instalação, injete os módulos ngImgCrop, ui.bootstrap e gumga.core na sua aplicação, como por exemplo:

angular.module('yourApp', ['ngImgCrop', 'ui.bootstrap', 'gumga.core']);
