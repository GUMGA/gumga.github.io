Diretiva - gumgaPattern

O componente GumgaPattern serve para validar expressões regulares de formulários.




Demonstração

Expressão não corresponde com o formato esperado

Exemplo de código

<form name="myForm" gumga-form>
  <label>CEP</label>
  <input type="text" name="cep" ng-model="cep" class="form-control" gumga-pattern="(\d{5})\-(\d{3})" id="cep" placeholder="99999-999">
  <p ng-show="myForm.cep.$error.pattern" class="text-danger">Expressão não corresponde com o formato esperado</p>
</form>

Atributos

Parâmetro Descrição
label Usado na integração com gumga.core:gumgaErrors para indicar em qual campo se encontra o erro. Se o atributo for omitido, a diretiva usará o atributo name do input.
