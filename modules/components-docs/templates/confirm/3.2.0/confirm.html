Diretiva - GumgaConfirm

O componente gumgaConfirm pode ser utilizado para interceptar um clique em algum elemento que contenha a diretiva ng-click, mostrar um modal de confirmação que, se o usuário clicar no botão de confirmação, a ação que está registrada no ng-click será executada.




Demonstração

Exemplo de código

<button type="button" ng-click="example()"
    gumga-confirm="Tem certeza que deseja realizar esta ação?"
    dismiss-button="Não"
    confirm-button="Sim"
    class="btn btn-info"
    confirm-button-class="btn btn-primary"
    dismiss-button-class="btn btn-default">
    Salvar
</button>
  
angular.module('app', ['ngImgCrop', 'ui.bootstrap', 'gumga.core'])
  .controller('ctrl', function($scope){

    $scope.example = function(){
        alert('Função executada');
    }

  })

Atributos

Parâmetro Uso Descrição
gumga-confirm obrigatório Atributo que deve ser uma string que irá conter o conteúdo da mensagem que será mostrado no modal.
dismiss-button opcional Atributo que deve ser uma string que irá conter o texto que irá no botão de cancelar.
confirm-button opcional Atributo que deve ser uma string que irá conter o texto que irá no botão de confirmar.
dismiss-button-class opcional Atributo que deve ser uma string que irá conter qual classe o botão de cancelar assumirá.
confirm-button-class opcional Atributo que deve ser uma string que irá conter qual classe o botão de confirmar assumirá.
size opcional Atributo que deve ter o tamanho da modal. Ex.: sm ou lg, por padrão será usado md.
