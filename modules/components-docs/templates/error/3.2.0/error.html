Diretiva - gumgaError

O componente gumgaError serve para mostrar mensagens de validações abaixo do input responsável pelo erro, o atributo gumga-required foi usada para demonstração para validação. A responsábilidade do gumgaError é apenas marcar os elementos que terão a mensagem de erro abaixo do campo.




Demonstração

Exemplo de código

<form name="Example" gumga-form>
  <label>Nome:</label>
  <input type="text" name="name" field="nome do cliente" class="form-control" ng-model="entity.name" gumga-error gumga-required>
</form>

Atributos

Parâmetro Tipo Descrição
field String String que terá o nome do input para retornar nas mensagens de erros.
