Service - gumgaWebStorage

O service GumgaWebStorage é utilizado para ajudar o desenvolvedor a trabalhar com o storage do Browser.


Exemplo de Uso

angular.module('app', ['ngImgCrop', 'ui.bootstrap', 'gumga.core'])
  .controller('ctrl', function($scope, GumgaWebStorage){

    GumgaWebStorage.setLocalStorageItem('nome', 'Mateus');

    console.log(GumgaWebStorage.getLocalStorageItem('nome')); //Mateus

    GumgaWebStorage.removeLocalStorageItem('nome');

    console.log(GumgaWebStorage.getLocalStorageItem('nome')); //NULL

  })

Método - setSessionStorageItem(key, value);

O método setSessionStorageItem aceita dois parâmetros key e value.

Parâmetros
Parâmetro Tipo Descrição
key String Qual o identificador do valor que será salvo.
value String/Number/Object Valor que será salvo no sessionStorage.


Método - getSessionStorageItem(key);

O método getSessionStorageItem aceita um parâmetro key.

Parâmetros
Parâmetro Tipo Descrição
key String Qual o identificador do valor que deseja recuperar.
Retorno do getSessionStorageItem: [String|Number|Object] - Retorna o valor que estiver na sessionStorage daquela key específica. Caso não encontre, o valor retornado é null.


Método - removeSessionStorageItem(key);

O método removeSessionStorageItem aceita um parâmetro key.

Parâmetros
Parâmetro Tipo Descrição
key String Qual o identificador do valor que será removido da sessionStorage.


Método - clearSessionStorage();

O método clearSessionStorage não aceita nenhum parâmetro e limpa a sessionStorage.


Método - getNumberOfItemsInSessionStorage();

O método getNumberOfItemsInSessionStorage não aceita nenhum parâmetro, e retorna o número de itens no localStorage.


Método - setLocalStorageItem(key, value);

O método setLocalStorageItem aceita dois parâmetros key e value.

Parâmetros
Parâmetro Tipo Descrição

value String/Number/Object Valor que será salvo no localStorage.


Método - getLocalStorageItem(key);

Parâmetros

Método - getNumberOfItemsInLocalStorage();
