Diretiva - gumgaRequired

O componente GumgaRequired serve para validar campos obrigatórios.




Demonstração

Campo obrigatório

Exemplo de código

<form name="myForm" gumga-form>
  <label>Nome</label>
  <input type="text" name="required" class="form-control" ng-model="required" gumga-required>
  <p ng-show="myForm.required.$error.required" class="text-danger">Campo obrigatório</p>
</form>

Atributos

Parâmetro Descrição
label Usado na integração com gumga.core:gumgaErrors para indicar em qual campo se encontra o erro. Se o atributo for omitido, a diretiva usará o atributo name do input.
