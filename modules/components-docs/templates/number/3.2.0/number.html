Diretiva - gumgaNumber

O componente GumgaNumber é usado para formatar valores numéricos, é composto por outros sub-componentes que serão apresentados a seguir.




Demonstração

Exemplo apenas números Exemplo máximo de números Exemplo máximo de casas decimais

Exemplo de código

<form name="Example">
  Exemplo apenas números
  <input type="text" ng-model="number.integer" gumga-number integer-number class="form-control">
  Exemplo máximo de números
  <input type="text" ng-model="number.iplaces" gumga-number integer-places="2" class="form-control">
  Exemplo máximo de casas decimais
  <input type="text" ng-model="number.dplaces" gumga-number decimal-places="2" class="form-control">
</form>

Atributos

Parâmetro Tipo Descrição
integer-number atributo ou classe Números inteiros.
integer-places atributo Número de casas do número.
decimal-places atributo Número de casas decimais após o ponto.
