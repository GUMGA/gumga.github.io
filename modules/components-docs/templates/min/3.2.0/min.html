Diretiva - gumgaMin [Date|Length|Number]

Obs: Todas as diretivas gumgaMin listadas abaixo, possuem o atributo "label" que é opcional, esse atributo é inserido na input, ele é usado na integração com gumgaErrors para indicar em qual campo se encontra o erro. Se o atributo for omitido, a diretiva usará o atributo name do input.



gumgaMinDate

O componente GumgaMinDate serve para validar datas mínimas em entradas de formulários.
Esta diretiva suporta apenas inputs do tipo date. O valor do atributo/diretiva é obrigatório e deve ser uma data.


Demonstração MinDate

Exemplo de código

<form name="myForm" gumga-form>
  <label>Data:</label>
  <input type="date" name="minDate" ng-model="maxDate" class="form-control" gumga-error gumga-min-date="2020-07-20">
</form>

gumgaMinLength

O componente GumgaMinLength serve para validar quantidades mínimas de caracteres em entradas de formulários.


Demonstração MinLength

Exemplo de código

<form name="myForm" gumga-form>
  <label>Nome</label>
  <input type="text" name="minLength" class="form-control" ng-model="minLength" gumga-error gumga-min-length="20" id="minLength">
</form>

gumgaMinNumber

O componente GumgaMinNumber serve para validar números mínimas em entradas de formulários.
Esta diretiva suporta apenas inputs do tipo number. O valor do atributo/diretiva é obrigatório e deve ser um número.


Demonstração MinNumber

Exemplo de código

<form name="myForm">
   <input type="number" name="minNumber" ng-model="minNumber" gumga-error gumga-min-number="20">
</form>
